## Run Locally

Clone the project

```bash
  git clone https://github.com/ruturajmengal2016/next-application.git
```

Go to the project directory

```bash
  cd next-application
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
